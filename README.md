# Vantage Landing Page — Option 3

Alternative landing page concept for Vantage.

## Tech stack

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Getting started

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` — start dev server
- `npm run build` — build production bundle
- `npm run preview` — preview production build
- `npm run lint` — run ESLint
- `npm run test` — run tests
